# 🤖 AI Prompts Used – Dev Detective Project

Below are the prompts used to build this project step-by-step using AI:


## 1. Project Idea
"Create a GitHub user search app using HTML, CSS, and JavaScript that fetches real user data using an API."


## 2. UI Design
"Create a responsive UI with a search bar and profile card using HTML and CSS."


## 3. Fetch API Integration
"Use JavaScript Fetch API to get GitHub user data based on username input."


## 4. Async/Await Logic
"Explain how to use async/await to fetch and display API data in JavaScript."


## 5. Profile Display
"Display GitHub user details like avatar, name, bio, join date, and profile link."


## 6. Error Handling
"Handle API errors and show 'User Not Found' if the username does not exist."


## 7. Loading State
"Show a loading message or spinner while fetching data from the API."


## 8. Repository Fetching
"Fetch and display the latest 5 repositories of a GitHub user."


## 9. Date Formatting
"Convert API date format into human-readable format like '25 Jan 2023'."


## 10. Responsive Design
"Make the entire layout responsive using Flexbox, Grid, and media queries."


## 11. Battle Mode Feature
"Create a feature to compare two GitHub users based on total stars."


## 12. Total Stars Logic
"Loop through repositories and calculate total stars using reduce()."


## 13. UI Highlight
"Highlight winner and loser cards using different colors."


## 14. Validation
"Validate GitHub username input using regex before making API call."


## 15. UX Improvement
"Add direct profile and repository links in battle mode for better user experience."


## 💡 Learning Outcome
- API Integration using Fetch
- Async/Await & Promises
- DOM Manipulation
- Responsive UI Design
- Real-world Project Building
